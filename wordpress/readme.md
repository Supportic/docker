# Wordpress Playground

Copy `.env.sample` -> `.env` and adjust variables.
Copy `.db.env.sample` -> `.db.env` and adjust variables.

http://localhost
http://localhost/wp-admin

**install**  
`make install`

**start**
`make start`

**enter container**  
`make shell`

**enter wpcli container**
`make wpcli`

**remove everything**  
`make erase`
